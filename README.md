# Project-Overview:
This project involved creating a sample restaurant website composed of several key pages: featuring a menu page, a page to make reservations, an about page, and a specials page. The site demonstrates practical implementation of ASP.NET Web Forms, server‑side controls, and structured navigation.
# Goal
Our goal is to deliver a complete, fully operational web application using ASP.NET Web Forms—one that showcases strong UI structure, smooth navigation, and practical real‑world functionality for a restaurant setting.
# How it Works:
The user can navigate through our website using the top navigation menu, including the Home, About Us, Menu, Specials, and Reservation pages. The menu has all item listings and prices available for all users to see. Users are also able to make a reservation with our restaurant, filling out their name, the number of guests in the party, and the selected date and time. Once submitted, we have a record of the reservation! The user may browse through weekly specials to prepare for their visit, and read about the history of the restaurant on the About Us page.
